Contributor to **cec2013lsgo**, a package for using the CEC'2013 Large-Scale Global Optimization benchmark in Python.

My work scaled the original benchmark from the fixed **D = 1000** setting to support arbitrary dimensions through deterministic, seed-based structural data generation.
